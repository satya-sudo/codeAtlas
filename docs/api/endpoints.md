# API Surface

## Auth

- `POST /auth/github`
- `GET /auth/me`

## Repositories

- `GET /repos`
- `POST /repos/connect`
- `GET /repos/:id`

## Analytics

- `GET /repos/:id/overview`
- `GET /repos/:id/hotspots`
- `GET /repos/:id/modules/ownership`
- `GET /repos/:id/modules/expertise`
- `GET /repos/:id/modules/bus-factor`
- `GET /repos/:id/contributors`
- `GET /repos/:id/graph`
